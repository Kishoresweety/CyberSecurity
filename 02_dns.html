About DNS:

DNS (Domain Name System) translates human-readable domain names like "example.com" into IP addresses like "93.184.216.34" that computers use to communicate. This hierarchical system ensures you don't need to memorize IP addresses.

Client: Initiates DNS query
Resolver: Recursive lookup
Root: Points to TLD
TLD: Top-level domain
Auth: Final answer
Packet: DNS query/response
